# Supply-Chain-Analytics-SQL
A SQL project that analyzes supply chain data using MySQL. It includes business-focused SQL queries for revenue, inventory, shipping, supplier performance, and quality analysis.
# Supply Chain Analytics using SQL

## Project Overview

This project analyzes a supply chain dataset using MySQL. The dataset was imported into MySQL Workbench, and SQL queries were written to analyze supplier performance, product revenue, shipping costs, inventory, and product quality.

The main objective of this project was to practice SQL concepts and perform business analysis using real-world data.

Tools Used

* MySQL Workbench
* SQL
* CSV Dataset

 Database

Database Name: `supply_chain`

Table Name: `supply_chain_data`
SQL Concepts Used

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* Aggregate Functions
* Subqueries
* CASE Statement
* Window Functions (RANK)
* Views

Business Questions Solved

* Top 5 highest revenue products
* Revenue by product type
* Average shipping cost by carrier
* Supplier performance analysis
* Revenue by location
* Products with high defect rate
* Products that passed quality inspection
* Products generating above-average revenue
* Product stock status
* Revenue ranking of products



Dataset

* Records: 100
* Columns: 24
